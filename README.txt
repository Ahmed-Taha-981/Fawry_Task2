Quantum Book Store - Simple Java Example
========================================

Files:
- Book.java
- PaperBook.java
- EBook.java
- ShowcaseBook.java
- Inventory.java
- QuantumBookstoreFullTest.java

How to compile:
  javac *.java

How to run the test:
  java QuantumBookstoreFullTest

This will demonstrate adding, removing, and buying books in the Quantum Book Store system.
